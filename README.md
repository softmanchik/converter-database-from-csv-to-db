# converter-database-from-csv-to-db
предлагаю ОТЛИЧНЫЙ конвертир который преврашает в таблиционый, бинарный базы данных .csv файлы или же .txt оно определяет все поля и разделители а затем создает индексатор FTS5 дефолтная табица это contacts
программа написано польностью на раст так что вы должны скачать rust(для самого кода) и cargo(для компиляций и т.д) раст вы можете скачать с https://rust-lang.org/ru/tools/install/ а cargo установиться вместе с ним это был тутор на Windows на линуксе все еше проше на убунту(based) - sudo apt install cargo, rust и все дальше можете уже пользоваться через Nvim
