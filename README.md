# converter-database-from-csv-to-db
Представляю удобный конвертер, который преобразует .csv и .txt файлы в табличную бинарную базу данных.

Программа автоматически определяет структуру данных: поля и разделители, после чего создает индексированную базу с использованием FTS5.
Таблица по умолчанию называется contacts.

Проект полностью написан на Rust, поэтому для работы потребуется установить:

Rust (для разработки)
Cargo (для сборки и управления зависимостями)

Скачать Rust можно здесь:
https://rust-lang.org/ru/tools/install/

Cargo устанавливается автоматически вместе с ним.

Установка

Windows:
Следуйте инструкции по ссылке выше.

Linux (например, Ubuntu-based):

sudo apt install cargo rustc

После установки можно сразу начинать пользоваться программой (например, через Nvim или любой другой редактор/CLI).
